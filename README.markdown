# BANANA CAT Website

A dynamic website with a moving galaxy background, "BANANA CAT" title, and a Minecraft icon.

## Features
- Animated galaxy background with colorful stars and nebula effect using HTML5 Canvas
- Centered "BANANA CAT" title with glowing effect
- Minecraft block icon next to the title
- Responsive design

## Setup
1. Clone the repository:
```bash
git clone https://github.com/yourusername/your-repo.git
```
2. Download a Minecraft block icon (e.g., diamond block) and name it `minecraft_icon.png` in the project folder.
3. Open `index.html` in a browser.

## Deployment
To deploy on GitHub Pages:
1. Push the code to your GitHub repository.
2. Go to repository Settings > Pages.
3. Select the main branch and save.
4. Your site will be live at `https://yourusername.github.io/your-repo`.

## Notes
- Replace `minecraft_icon.png` with an actual Minecraft block image (e.g., from https://minecraft.wiki).
- Update `yourusername` and `your-repo` in the README with your GitHub details.
